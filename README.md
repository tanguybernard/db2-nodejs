# db2

## Launch db2 instance

    docker-compose up


## Usefull Command


### Connection to database

    db2 connect to testdb

### To see columns and data types of a table

    db2 describe table <schema>.<table_name>

### To retrieve values form a table

    db2 "select * from <schema>.<table>"
